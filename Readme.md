# SurgeSync

The SurgeSync app simplifies the deployment process for React projects by enabling users to deploy any React application with just a single click. Whether it's a personal portfolio, a company website, or a side project, users can instantly host their React apps without the hassle of manual configurations.

## Tech Stack

- React: A JavaScript library for building user interfaces.
- Node.js: A runtime environment for executing JavaScript code server-side.
- Express.js: A web application framework for Node.js, providing a robust set of features for web and mobile applications.
- Kafka: A distributed event streaming platform for building real-time data pipelines and streaming applications.
- Socket.IO: A library for real-time web applications, enabling bidirectional communication between clients and servers.
- HTTP Proxy: A proxy server for routing HTTP requests between clients and servers.
- Prisma: A modern database toolkit for Node.js and TypeScript, providing an ORM layer for database access.
- PostgreSQL: A powerful, open-source relational database management system.
- ClickHouse: An open-source column-oriented database management system for analytics and data warehousing.
- AWS (Amazon Web Services): A comprehensive cloud platform offering a wide range of services for computing, storage, networking, and more.
